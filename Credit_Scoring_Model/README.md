# 💳 Credit Scoring Prediction App

## 📌 Project Overview
This project is a Machine Learning-based Credit Scoring Application developed as part of the CodeAlpha Internship.

The application predicts whether a loan should be approved or rejected based on customer financial and credit details.

---

## 🎯 Objective
To build a classification model that evaluates an individual's creditworthiness using historical financial data.

---

## 🧠 Machine Learning Approach
- Algorithm Used: Random Forest Classifier
- Data Processing:
  - Handling categorical variables using one-hot encoding
  - Feature engineering (loan-to-income ratio)
- Model Evaluation:
  - Accuracy Score
  - Probability-based risk prediction

---

## 🖥️ Application Features
- User-friendly UI built with Streamlit
- Input sections:
  - Customer Details
  - Financial Details
  - Credit Details
- Output includes:
  - Loan Approval / Rejection
  - Risk Percentage
  - AI-based Explanation
  - Risk Analysis (Donut Chart)
  - Smart Insights
